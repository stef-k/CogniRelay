# MCP 216 Convergence Audit

This document is normative for slice 1 of `#216`.

It records the hardened `#216` MCP convergence target and the exact canonical matrix required by the issue body. Where older descriptive MCP prose in the repo differs from this audit, this audit controls.

## Convergence Target

- Target: MCP `2025-11-25` Streamable HTTP
- Temporary posture for `#216`: `POST /v1/mcp` only
- Temporary deferred posture for the GET transport path: `GET /v1/mcp = 405 + Allow: POST`
- Supplemental metadata endpoint only: `GET /.well-known/mcp.json`
